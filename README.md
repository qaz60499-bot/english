# 英语星球 1.0.0

面向中文母语学习者的本地优先英语学习 PWA。项目基于 React、TypeScript、Vite、Dexie、Zustand 与 PWA 技术，无需账号、后端、API 或第三方学习 SDK。

## 产品能力

- 基础、四级、六级、雅思、托福共 288 节循序渐进课程。
- 3456 道课程练习，覆盖 17 种题型。
- 词汇、核心句型、场景表达、语法、发音、文化故事与复习系统分离维护。
- 美式英语、英式英语与自动口音选择。
- 课程进度、错题、记忆强度、设置、导入导出全部保存在本地 IndexedDB。
- 支持手机和桌面布局、PWA 离线缓存、明暗主题、键盘操作和减少动画。

## 内容规模

- 2304 个主动词汇与高频词块

## 本地开发

```bash
npm ci
npm run content:generate
npm run validate:content
npm run typecheck
npm run test
npm run build
npm run test:e2e
```

开发服务器：

```bash
npm run dev -- --port 5180
```

## 部署

先构建静态文件：

```bash
npm run build
```

Cloudflare Pages：

- 构建命令：`npm run build`
- 构建输出目录：`dist`
- 若手动上传，上传 `dist` 目录内的全部文件，确保 `index.html` 位于根目录。

GitHub Pages：

- 构建命令：`npm run build`
- 发布目录：`dist`
- 如仓库不是根域名站点，需要在 `vite.config.ts` 设置对应 `base` 路径后重新构建。

## 数据与隐私

- 学习记录仅保存在当前浏览器本地。
- 录音只用于本地回听，不上传服务器。
- 清除浏览器数据或更换设备不会自动同步学习记录。
- 导入数据会先经过结构和范围校验，异常文件会被拒绝。

## 已知边界

- 浏览器朗读质量取决于设备安装的英语语音。
- 发音练习提供标准音、录音回听和自评，不提供音素级自动评分。
- 自动内容校验能检查结构、重复、映射和范围问题，但仍建议人工抽查学习内容质量。
